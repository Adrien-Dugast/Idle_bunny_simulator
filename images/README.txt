This folder is for rabbit and grass image files.

To add your pictures:
- save your image files here, for example:
  - meadow.png
  - forest.png
  - snow.png
  - golden.png
  - grass.png

Then, open script.js and update the imagePath values in the rabbitTypes array if you use different names or subfolders.

Example:
  imagePath: 'images/meadow.png'

The game will use these image paths for the rabbit cards as soon as the files exist.
